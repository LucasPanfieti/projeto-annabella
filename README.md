# Projeto CSS - Site Anna Bella

Este é um projeto desenvolvido durante o curso da Udemy **"Domine Web - 20 Cursos - HTML5, CSS3, SASS, Bootstrap, JS, ES6, PHP, OO, MySQL, JQuery, MVC, APIs, IONIC e muito mais"**, como parte prática do aprendizado de estruturação de páginas web.

Trata-se de um site institucional simples e estático de uma modelo fictícia chamada **Anna Bella**, com foco na apresentação de biografia, campanhas publicitárias e informações de contato.

## Sobre o projeto
- Página estruturada com HTML5.
- Estilização com CSS externo.
- Navegação entre páginas por meio de links internos.
- Utilização de imagens (capa, fundo e fotos das campanhas).
- Layout com divisões básicas utilizando <div> e classes/IDs personalizados.
- Projeto didático, sem uso de JavaScript ou frameworks.
